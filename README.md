## Run Locally

Clone the project

```bash
  git clone https://github.com/Shakti10128/code_editor_assignment
```

Go to the project directory

```bash
  cd code_editor_assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
